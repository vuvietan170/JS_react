# keyboard Shortcut
_Ctrl + G: Group object (text, image, shape, frame)  != Ctrl shift G : ungroup
_Ctrl Z , Ctrl Y : undo and redo
_ctrl C, Ctrl V: copy and paste 
_ Ctrl D : duplicate 

Ctrl shift k : create component 
_ Shift A : Add Auto layout
_ Shift 1 : Zoom to fit

_ F : Frame tool
_ T : Text tool
_ P : pen (draw, create vector)
_ R : Rectangle 
_ C : comments tool
